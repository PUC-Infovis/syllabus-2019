La sintaxis de un selector de CSS es la siguiente:

selector { atributo: valor; }

Donde el selector corresponde al elemento al que se le aplicará el estilo. El atributo es la característica estética que será cambiada, mientras que el valor indica en qué medida será aplicado el cambio de estilo.

El selector puede ser una etiqueta HTML, una clase o un id. Dependiendo del selector que se quiere utilizar, la sintaxis varía de la siguiente forma: 

                  
/* selector de una etiqueta */
div { atributo: valor; }
p { atributo: valor; }
h1 { atributo: valor; }

/* selector de una clase */
.clase1 { atributo: valor; }
.clase2 { atributo: valor; }
.clase3 { atributo: valor; }

/* selector de un id */
#id1 { atributo: valor; }
#id2 { atributo: valor; }
#id3 { atributo: valor; }

Además, se puede especificar más aún el selector. Por ejemplo, para seleccionar sólo los párrafos que tengan una clase específica, la sintaxis es: 

p .clase { atributo: valor; }

Supongamos que deseamos mostrar 4 imágenes dentro de un mismo cuadrado, cada con las mismas dimensiones. CSS tiene la opción de ordenar contenido mediante grillas. Por ejemplo, para lograr lo antes planteado mediante CSS y grid el código necesario es: 

.contenedor-imagenes { 
  display: grid;
  grid-template-columns: 100px 100px;
  grid-template-rows: 100px 100px;
}

.contenedor-imagen {
  width: 100%;
  height: 100%;
}

.contenedor-imagen img {
  width: 100%;
  height: 100%;
}

La línea display: grid activa el funcionamiento de grilla del elemento. El atributo grid-template-columns determina cuantas columnas tendrá la grilla, y de qué tamaño serán, mientras que grid-template-rows hace lo mismo con las filas. En el ejemplo anterior, se determina que cada columna tendrá un ancho de 100px, y cada fila una altura de 100px. Pero estas dimensiones se pueden determinar de distintas formas: 

// Dimensiones estáticas
.grilla { 
  display: grid;
  grid-template-columns: 100px 100px;
  grid-template-rows: 100px 100px;
}

// Dimensiones relativas
// fr = fraccion, es decir, se repartirá el espacio en dos fracciones iguales
.grilla { 
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-template-rows: 1fr 1fr;
}

// Dimensiones responsivas y relativas al tamaño de la ventana
.grilla { 
  display: grid;
  grid-template-columns: 20vw 20vw;
  grid-template-rows: 20vh 20vh;
}

// Dimensiones responsivas y estáticas
// Tendrán un tamaño mínimo de 100px y un máximo
// del 20% del largo de la ventana del navegador
.grilla { 
  display: grid;
  grid-template-columns: minmax(100px, 20vw) minmax(100px, 20vw);
  grid-template-rows: minmax(100px, 20vw) minmax(100px, 20vw);
}

// Dimensiones adaptables
// La columna del medio tendrá 200px de ancho, mientras que 
// las otras dos rellenarán el resto del tamaño disponible
.grilla { 
  display: grid;
  grid-template-columns: auto 200px auto;
}

Para definir en qué espacio irán los elementos dentro de la grilla se utilizan los atributos grid-column y grid-row. La sintaxis es la siguiente: si se quiere ubicar un elemento en la la segunda columna, y cuarta fila el código sería: 

.elemento { 
  grid-column: 2/3;
  grid-row: 4/5;
}

Cabe notar que los elementos ubicados dentro de la grilla deben estar dentro el tag asociado a la grilla que los contiene.

Al igual que grid, flex es una opción de display que permite ordenar los elementos dentro de un espacio. Está pensado para que este orden sea responsivo, lo que lo vuelve una opción muy útil cuando queremos que los elementos se encuentren en una posición sin importar las dimensiones del navegador.

Por ejemplo, para mostrar imágenes de 100x100px en una columna, siempre utilizando el centro horizontal de su contenedor, el código CSS es: 

.contenedor { 
  width: 100%;
  display: flex;
  flex-direction: column;
  align-items: center; 
}

.contenedor img {
  width: 100px;
  height: 100px;
}

La línea display: flex activa el modo flex dentro del elemento. El atributo flex-direction indica en qué dirección se alinearán los elementos dentro del contenedor; pueden alinearse como columna o fila. Por otro lado, align-items corresponde a la alineación que tendrán los elementos; pueden ubicarse al final del contanador, al inicio, al centro, etc. Otro atributo importante es justify-content, el cual indica cómo se reparten el espacio los distintos elementos. Algunos ejemplos de justify-content: 

.ej1 { 
  display: flex;
  flex-direction: row;
  justify-content: center;
}

.ej2 { 
  display: flex;
  flex-direction: row;
  justify-content: space-between;
}

.ej3 { 
  display: flex;
  flex-direction: row;
  justify-content: space-around;
}
